# Personal website
My personal website where I showcase selected projects like websites or tools i have built and talk about how they came into existence and what tech is used and what my role is in them.

**Live Site:** [viktorhollanders.com](https://viktorhollanders.com)

## Built With
* **Framework:** vite + react
* **Styling:** tailwind
* **Deployment:** vercel

## Getting Started
Needs
*  Node.js 22.x or higher
*  pnpm (preferred package manager)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/viktorhollanders/viktorhollanders-web
   cd personal-website
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Run the development server:
   ```bash
   pnpm dev
   ```

Open [http://localhost:5173](http://localhost:5173) in your browser to view the local build.
